nG配列変換計算機

入力はψ(a,b,c,d,...,x,y,z)またはΨ_n(b)(nは自然数)の形式で入力してください。変数の個数は固定でお願いします。
略記として多変数は、1 := ψ(0,0,...,0), n := 1 + 1 + ...(n個の1), ω := ψ(0,0,...,1), Ω := ψ(0,0,...,1,0), I := ψ(0,0,...,1,0,0)
2変数は、1 := Ψ_0(0), n := 1 + 1 + ...(n個の1), ω := Ψ_0(1), Ω := Ψ_1(0)が使用可能です。
また、ψはpで、ΨはPで、ωはwで、ΩはWで代用可能です。
